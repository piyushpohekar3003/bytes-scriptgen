HOW TO RUN:
  1. Double-click "BytesScriptGen.command"
  2. Browser opens automatically
  3. That's it!

FIRST RUN:
  Takes ~30 seconds to install Python packages.
  After that, starts instantly.

REQUIREMENTS:
  - macOS with Python 3.8+ (pre-installed on most Macs)
  - Internet connection (for news fetching + AI)

USE YOUR OWN API KEY (optional):
  1. Create a file called ".env" in this folder
  2. Add: ANTHROPIC_API_KEY=sk-ant-api03-your-key-here
  3. Restart the app

TO STOP:
  Press Ctrl+C in the terminal window, or just close it.
